# ₿ Bitcoin 4-Year Cycle Model

A minimal single-page app visualizing Bitcoin's halving cycle pattern.

## 📊 Features

- **Halving Countdown** — Real-time countdown to the next halving event
- **Current Phase** — Auto-detects the current phase in the cycle with a progress bar
- **Full Timeline** — Displays all 5 phases across the complete 48-month cycle

## 🔄 Halving T0 Model

Using the halving month as T0:

| Phase | Period | Description |
|---|---|---|
| Consolidation | T+0 ~ T+12 | Market digests supply shock; uptrend begins late in the phase |
| Main Rally + Top | T+12 ~ T+18 | Most likely to see the main rally and cycle top (avg. T+16) |
| Weakening | T+18 ~ T+24 | Trend weakens; historical turning point into bear market |
| Bear Bottom | T+24 ~ T+30 | Historical bear bottom window (statistical avg. T+28) |
| Recovery | T+30 ~ T+48 | Slow recovery, preparing for the next halving cycle |
