# ₿ 比特币四年周期模型

一个简洁的单页应用，可视化展示比特币减半周期规律。

## 📊 功能

- **减半倒计时** — 实时显示距下次减半的剩余时间
- **当前阶段** — 自动判断当前处于周期中的哪个阶段，附带进度条
- **完整时间线** — 展示 48 个月完整周期的 5 个阶段

## 🔄 减半 T0 模型

以减半月份为 T0：

| 阶段 | 时间 | 说明 |
|---|---|---|
| 震荡阶段 | T+0 ~ T+12 | 市场消化供给变化，后期上升启动 |
| 主升浪 + 见顶 | T+12 ~ T+18 | 最容易出现主升浪，见顶均值 T+16 |
| 高位转弱 | T+18 ~ T+24 | 趋势转弱，转向熊市的转弯阶段 |
| 熊底窗口 | T+24 ~ T+30 | 历史熊底窗口，统计均值 T+28 |
| 修复期 | T+30 ~ T+48 | 缓慢修复，准备进入下一轮减半 |

## 🚀 使用

纯静态 HTML，无需任何依赖，直接打开即可：

open index.html
